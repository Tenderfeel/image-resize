# Image Resize Task

Sharpによって画像をリサイズするGulpタスクのサンプル。  


## Command
Node.js v14.17.5(lts/fermium) 以上で動作、v16.7.0 推奨

```bash
npm install
npm start

# or 

yarn install
yarn start
```
